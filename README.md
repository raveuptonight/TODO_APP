# パチンコ TODO - ドパガキ専用

報酬系を刺激しまくるパチンコ風TODOアプリ

## セットアップ

### 1. 依存関係インストール

```bash
npm install
```

### 2. 音声ファイルのダウンロード

[牙狼公式サイト](https://www.sansei-rd.com/p_garo_archive/download/sound.html)から音声ファイルをダウンロードして、`public/sounds/` に配置してください。

### ファイル名の対応表

`src/SoundManager.js` の `soundFiles` オブジェクトを編集して、ダウンロードしたファイル名に合わせてください。

推奨する音声の割り当て：

| 用途 | おすすめの音声 |
|------|---------------|
| `holdIn` | ガロ保留音（赤） |
| `holdChange` | 変動開始時エンブレム完成音（チャンス） |
| `holdRainbow` | 変動開始時エンブレム完成音（激アツ） |
| `gekiatsu` | ガロフラッシュ音（赤） |
| `impact` | インパクト予告（牙狼斬馬剣） |
| `reach` | 牙狼SPリーチ/タイトル（赤） |
| `spTransition` | F.O.G.完成音 |
| `yakumono` | 牙狼斬馬剣飛び出し音 |
| `swordPush` | 牙狼剣押下成功音 |
| `toukaku` | F.O.G.完成音（大当り時） |
| `ooatari` | BATTLE BONUS 3000獲得音 |
| `vEntry` | 連続予告後タイトル（VICTORY） |
| `kakuhenStart` | THROUGHOUT BONUS継続音 |
| `roundStart` | 昇格演出中/図柄昇格成功音 |
| `spReachBgm` | CLIMAX BATTLE BGM |
| `feverBgm` | 金色コレクションBGM |

### 3. 起動

```bash
npm run dev
```

ブラウザで http://localhost:5173 を開く

## 使い方

1. タスクを入力して「追加」
2. タスクをタップすると保留に追加（最大4個）
3. 保留が自動で消化され、リーチ演出 → 大当たり判定
4. 大当たりでタスク完了！ポイント獲得！
5. 7 or V 揃いで確変突入（当選率UP）

## 演出

- **保留変化**: 青 → 緑 → 赤 → 金 → 虹
- **激アツ予告**: 赤フラッシュ＋画面シェイク
- **SPリーチ発展**: 役物演出、ボタン演出
- **大当たり**: 虹エフェクト、ファンファーレ
- **確変**: 継続率表示（65%/82%）

## 音声ファイルがない場合

`SoundManager.js` が自動でWeb Audio APIによる代替音を生成するので、音声ファイルがなくても動作します。ただし本物の音の方が圧倒的に脳汁出ます。
